About Us

Our Breaking Bad File Encryption App is a Software Engineering Project which allows users to store sensitive data on their mobile phones without having to worry about it being leaked, even if they have lost their mobiles. This project concentrates on securing data on mobile phones by storing it in an encrypted form. The data will be encrypted using some sequence and the key will be stored on the server. If at any time the user loses their mobile phones, the user can send a report to the server which then destroys the respective key so the data on the mobile can never be decrypted.

Creators of Breaking Bad File Encryption: - Megan Williams - Kara Suraj - Anthony Hosang - Akash Manohar
